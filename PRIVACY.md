# Privacy Policy

GitHub Isometric Contributions is a browser extension that renders GitHub contribution graphs as isometric 3D pixel art.

## Data Collection

This extension does not collect, store, or transmit any personal data. It does not use analytics, tracking, or any third-party services.

## Data Usage

The extension reads contribution data directly from the GitHub profile page you are viewing in order to render the isometric visualization. This data is processed entirely within your browser and is never sent anywhere.

## Storage

The extension stores a single user preference (2D/3D view toggle) locally in your browser using the browser's built-in storage API. This data never leaves your device.

## Contact

If you have questions about this privacy policy, you can open an issue at https://github.com/jasonlong/isometric-contributions/issues.
